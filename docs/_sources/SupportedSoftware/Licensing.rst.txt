Licensing
===========


You must have a Matlab license to use the Matlab application on the UW-Madison Cryo-EM HPC. More information about Matlab can be found on the `Matlab website <https://www.mathworks.com/products/matlab.html>`_.                        


You must have an SBGrid license to use the SBGrid software on the UW-Madison Cryo-EM HPC. More information about SBGrid can be found at `SBGrid.org <https://sbgrid.org>`_.
